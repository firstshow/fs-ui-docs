介绍

fs-ui轻量、可靠的移动端 Vue 组件库

特性

  • 50+个经过付呗线上业务检验的组件、持续递增中...
  • 单元测试覆盖率超过 85%
  • 细致、漂亮的 UI
  • 支持 babel-plugin-import
  • 高质量、功能丰富；
  • 友好的 API ，自由灵活地使用空间；
